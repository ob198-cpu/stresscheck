# ストレスチェック受検システム 操作ガイド

## これは何か

`server.js` は、ストレスチェック受検システムを動かすためのプログラム本体です。

WordやPDFのように開いて読むファイルではありません。  
起動すると、スマホやPCのブラウザから回答できる画面を表示します。

## 使い方

### 1. システムを起動する

以下のファイルをダブルクリックします。

```text
C:\ストレスチェック\stresscheck-system\起動.bat
```

黒い画面が開き、自動でブラウザが開きます。

次のような表示が出れば起動中です。

```text
== Started
http://127.0.0.1:8787
```

黒い画面は閉じないでください。閉じるとシステムも止まります。

### 2. ブラウザで開く

通常は自動で開きます。開かない場合だけ、ブラウザで以下を開きます。

```text
http://127.0.0.1:8787
```

### 3. 画面の意味

| 画面 | 用途 |
|---|---|
| 従業員デモ回答 | 従業員がスマホで回答する画面 |
| 会社担当者画面 | 会社が受検済み/未受検と集団分析を見る画面 |
| 実施者画面 | あなたが個人結果・高ストレス判定・同意状況を見る画面 |
| QRコード付き受検案内 | 受検者ごとの本人専用URLとQRコードを印刷する画面 |
| 社内メール文 | 社内メールで本人専用URLを配布するための文面 |

## 受検者に回答してもらう方法

実施者画面から、会社ごとの `QRコード付き受検案内を作成` を開きます。

そこで受検者ごとの本人専用URLとQRコードが表示されます。

配布方法は2つあります。

### QRコードで配布する

1. 実施者画面を開く
2. `QRコード付き受検案内を作成` を押す
3. `印刷する` を押す
4. 受検者本人に、自分の名前が書かれた案内だけを渡す
5. 受検者はスマホでQRコードを読み込んで回答する

### 社内メールで配布する

1. 実施者画面を開く
2. `QRコード付き受検案内を作成` を押す
3. `社内メール文を見る` を押す
4. メール文を使って、本人専用URLを受検者に送る

注意: 本人専用URLは配り間違えると別人の回答画面に入れてしまいます。配布時は氏名を必ず確認してください。

## 大事なポイント

会社担当者画面では、個人結果・点数・高ストレス判定は見えません。

個人結果を見られるのは、本人と実施者であるあなたです。

## 終了方法

起動時に開いた黒い画面を閉じると、システムは停止します。

## うまく開けない場合

1. `起動.bat` をダブルクリックしたか確認する
2. 黒い画面を閉じていないか確認する
3. ブラウザで `http://127.0.0.1:8787` を開く
4. それでも開けない場合は、黒い画面に表示されたエラーを確認する

## 本番利用前に必要なこと

このシステムはMVPです。本番で使う前に以下が必要です。

- HTTPS化
- 正式なログイン機能
- 管理者パスワードの変更
- データ暗号化
- 厚労省方式に沿った設問・判定基準の最終確認
- 契約書類の専門家レビュー
