# InfoM Googleフォーム生成スクリプト

`createStressCheckGoogleForm()` または `createStressCheckGoogleFormOnly()` を実行すると、職業性ストレス簡易調査票57項目のGoogleフォームを作成します。

Googleフォームで回答を集める場合、回答スプレッドシートをCSVでダウンロードし、ローカル管理画面の `GoogleフォームCSV取込` から読み込みます。Google API連携を前提にしないため、OAuthのブロックで止まりにくい運用です。

取込側は、設問見出しが `A1. ...`、`B1. ...` のように始まっていれば57項目を読み取れます。見出しだけ確認したい場合は、管理画面の `CSV見出しテンプレートを保存` を使ってください。

運用上の注意:

- 会社担当者をフォーム編集者にしない。
- 回答スプレッドシートを会社に共有しない。
- 会社へ共有するのは受検済み・未受検と集団分析のみ。
- 個人結果通知はGoogleフォームの自動返信では行わない。
- 本人専用IDと従業員本人の対応表は別管理にする。
